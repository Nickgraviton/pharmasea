API

Στον προγραμματισμό του υπολογιστή, μια διεπαφή προγραμματισμού εφαρμογών (API) είναι ένα σύνολο ορισμών υπορουτίνας , πρωτόκολλα επικοινωνίας και εργαλεία για την οικοδόμηση λογισμικού. Σε γενικές γραμμές, είναι ένα σύνολο σαφώς καθορισμένων μεθόδων επικοινωνίας μεταξύ των διαφόρων συνιστωσών. Ένα καλό API διευκολύνει την ανάπτυξη ενός προγράμματος υπολογιστή, παρέχοντας όλα τα δομικά στοιχεία, τα οποία στη συνέχεια συνθέτει ο προγραμματιστής. Ένα API μπορεί να είναι για ένα web-based σύστημα, λειτουργικό σύστημα, σύστημα βάσης δεδομένων, υλικό υπολογιστή ή βιβλιοθήκη λογισμικού. Μια προδιαγραφή API μπορεί να λάβει πολλές μορφές, αλλά συχνά περιλαμβάνει προδιαγραφές για ρουτίνες, δομές δεδομένων, κλάσεις αντικειμένων, μεταβλητές ή απομακρυσμένες κλήσεις. Τα POSIX, το API των Windows και το ASPI είναι παραδείγματα διαφορετικών μορφών API. Η τεκμηρίωση για το API παρέχεται συνήθως για να διευκολύνει τη χρήση και την εφαρμογή.

Εφαρμογή

Στο πλαίσο της κατασκευής μιας αποτελεσματικής πλατφόρμας αναζήτησης φαρμάκων αποφασίσαμε να να κάνουμε χρήση API για να παρέχουμε σε σας τους πελάτες μας ένα σύγχρονο και αποδοτικό σύστημα όχι μονο λειτουργικό αλλά και απλό στην χρήση και αισθητικά όμορφο.Επίσης για τους πρόθυμους developers τα API που χρησιμοποιούμε μπορείτε εύκολα να τα δοκιμάσετε αλλά και να τα εμπλουτίσετε καθώς προσφέρονται χωρίς πληρωμή απαιτώντας απλώς σεβασμό στα πνευματικά δικαιώματα και λογαριασμό για να λάβετε public key.

Χρησιμοποιούμενα API

RESTful

Για την εύκολη αναζήτηση του χρήστη και τα αιτήματα του έχουμε κάνει χρήση του RESTful API μεσω του οποίου ο χρήστης μπορεί να κάνει αιτήματα στην βάση δεδομένων μας για προιόντα και καταστήματα και να λαμβάνει δυναμικά αποτελέσματα.Αυτό κατορθώνεται με endpoints στα οποία γίνονται queries και επιστρέφονται αποτελέσματα.

Bing Maps

Επίσης για την περιήγηση του χρήστη σε ενσωματωμένο χάρτη έγινε χρήση του Bing Maps API για την αναζήτηση συντεταγμένων απο διευθύνσεις (geocoding) αλλά και για το αντίστροφο (reverse geocoding).Ετσι ο χρήστης μπορεί να βλέπει τα μαγαζιά στον χάρτη και να επιλέγει ανάλογα.

mapbox API(leaflet plugin)

Για την άντληση δεδομένων τοποθεσίων και τον σχηματισμό και την ενσωμάτωση χαρτών χρησιμοποιήσαμε το mapbox API και το leaflet plugin της javascript για διαδραστικούς-mobile friendly χάρτες και την δυνατότητα να επεκτείνουμε την ιδέα μας σε κινητές εφαρμογές στο μέλλον
